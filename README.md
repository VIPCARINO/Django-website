AI Blog Website — README

Overview

This project is a modern blog and content management website built using Django.
It provides users with a clean and responsive platform for reading, creating, and managing blog content efficiently.

The system was designed with scalability, performance, and user experience in mind while implementing full-stack web development concepts.

Installation

Clone the Repository

git clone https://github.com/VIPCARINO/Django-website.git

cd project-name

Create Virtual Environment

python -m venv venv

Activate Environment

Windows

venv\Scripts\activate

Linux/Mac

source venv/bin/activate


---

Install Dependencies

pip install -r requirements.txt


---

Apply Migrations

python manage.py migrate


---

Create Superuser

python manage.py createsuperuser


---

Run Development Server

python manage.py runserver

Visit:

http://127.0.0.1:8000/
